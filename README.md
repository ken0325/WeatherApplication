# Weather App for Hong Kong

## Overview

This project is a mobile application designed to provide users with accurate and up-to-date weather information for Hong Kong. Developed in Java for Android, the application utilizes the Hong Kong Observatory's open API and the Open-Meteo API to fetch real-time weather data, forecasts, and other relevant weather information.

## Features

The application includes the following features:

- **Current Weather Information**: Displays the current weather conditions based on the user's location.
- **Local Weather Forecast**: Provides short-term and long-term weather forecasts specific to Hong Kong.
- **9-Day Weather Forecast**: Offers a comprehensive 9-day weather forecast for better planning.
- **Temperature Converter**: Allows users to convert temperatures between Celsius and Fahrenheit.
- **Weather Alerts**: Notifies users of any weather-related disaster warnings issued by local authorities.

## Technologies Used

- **Programming Language**: Java
- **Platform**: Android
- **APIs**:
  - Hong Kong Observatory Open API
  - Open-Meteo API
- **Data Format**: JSON

## Project Structure

- **/src**: Contains the main source code for the application.
- **/res**: Contains resources such as layouts and images.
- **/AndroidManifest.xml**: Configuration file for the Android application.
